# NYC Urban Heat Island Visualization

An interactive Streamlit dashboard exploring the Urban Heat Island effect in New York City.

## Features

- **Explore UHI Effect**: Scientific overview of contributing factors, day vs. night temperature differences, and interactive choropleth map of heat vulnerability by neighborhood
- **Inequality & UHI**: Connection between historical redlining and present-day temperature disparities, with health outcome data
- **Mitigations & Solutions**: Evidence-based cooling strategies and links to NYC programs

## Data Sources

- NYC Department of Health Heat Vulnerability Index (HVI)
- Hoffman, Shandas & Pendleton (2020) - Redlining and urban heat study
- NYSERDA temperature data
- NYC Panel on Climate Change 4th Assessment (2024)
- EPA Heat Island Effect documentation

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
